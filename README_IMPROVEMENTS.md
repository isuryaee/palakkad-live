## 💡 Customization Guide

### Change Colors
Edit `app/globals.css` - Update Tailwind color references

### Change Fonts
Edit `app/layout.tsx` - Update Google Fonts import

### Add Categories
Edit `app/category/[slug]/page.tsx` - Add to CATEGORIES_MAP

### Add Locations
Edit `app/location/[slug]/page.tsx` - Add to LOCATIONS_MAP

### Modify Admin Stats
Edit `app/desk/page.tsx` - Update data structure
